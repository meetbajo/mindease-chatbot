# MindEase – Mental Health Chatbot 🧠💜

An AI-powered mental health chatbot built with **Streamlit** for a college assignment (Proof of Concept).

## 🚀 Live Demo
[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://mindease-mental-health.streamlit.app)

## ✨ Features

| Feature | Description |
|---------|-------------|
| 💬 **AI Chat (Elara)** | Empathetic chatbot responding to anxiety, stress, depression, loneliness & more |
| 📊 **Mood Tracker** | Weekly mood chart with stats and daily logging |
| 🫁 **Breathing Exercises** | Guided 4-7-8, Box Breathing, and Belly Breathing with animated timer |
| 📓 **Journal Prompts** | 12 rotating CBT-based reflective writing prompts |
| 🧩 **CBT Techniques** | Interactive thought record and evidence-based exercises |
| 🧘 **Mindfulness Guide** | 5-4-3-2-1 grounding, body scan, and visualization techniques |
| 🆘 **Crisis Support** | India (iCall, NIMHANS, Vandrevala) and global helplines |

## 🛠️ Tech Stack

- **Python** + **Streamlit**
- **Pandas** for data handling
- Custom CSS (glassmorphism dark theme, Google Fonts)

## 📦 Installation

```bash
git clone https://github.com/YOUR_USERNAME/mindease-chatbot.git
cd mindease-chatbot
pip install -r requirements.txt
streamlit run app.py
```
